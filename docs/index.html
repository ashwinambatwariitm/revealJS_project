def roi(gain, cost):
    """Compute simple ROI as a percentage."""
    return ((gain - cost) / cost) * 100

Quick Demo: Calculate NPV (JavaScript)

Open the console (or run this snippet in a dev tools console).


function npv(rate, cashflows) {
  return cashflows.reduce((acc, cf, i) => acc + cf / Math.pow(1 + rate, i), 0);
}

console.log(npv(0.08, [-100000, 30000, 40000, 50000, 60000]));
