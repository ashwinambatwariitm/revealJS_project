Python ROI Example


def roi(gain, cost):
    return ((gain - cost) / cost) * 100

print(roi(120000, 100000))
ROI is a key profitability metric.

Financial Formulas

NPV:

\( \text{NPV} = \sum_{t=0}^{T} \frac{C_t}{(1+r)^t} \)

ROE:

\( \text{ROE} = \frac{\text{Net Income}}{\text{Equity}} \times 100\% \)

Used for valuation & profitability.

JavaScript NPV Demo


function npv(rate, cashflows) {
  return cashflows
    .reduce((acc, cf, i) => acc + cf / Math.pow(1 + rate, i), 0);
}

console.log(npv(0.08, [-100000, 30000, 40000, 50000, 60000]));
Console output shows computed NPV.
